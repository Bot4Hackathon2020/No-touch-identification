#### **项目名称**：

WearMask!—智能口罩检测门禁系统

## 安装步骤

___

1. git clone 目录
2. 配置node环境
3. 配置python环境
4. npm install 安装electron依赖
5. pip install安装python所需的库
6. npm start开始运行
